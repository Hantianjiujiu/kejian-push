# 招聘人数竞速动画 — 设计规格

## 概述

Remotion 动画视频，展示阿里巴巴、腾讯、华为 2021-2025 年招聘人数的动态柱状图竞速（bar chart race），呈现三强拉锯战态势，36氪/第一财经风格。

## 视频规格

- 分辨率：2560×1440
- FPS：30
- 总时长：约 75 秒（2250 帧）
- 格式：JPEG（统一项目配置）

## 整体结构

| 章节 | 时间 | 内容 |
|------|------|------|
| 开场 | 0-3s | 标题"科技巨头人才争夺战"淡入，三家 Logo 依次闪现 |
| 2021 | 3-15s | 阿里领头、腾讯紧随，华为承压，"后疫情抢人大战" |
| 2022 | 15-27s | 阿里继续扩招拉开差距，"史上最大规模校招"标记 |
| 2023 | 27-42s | 转折之年——阿里收缩，华为反弹，"麒麟回归"标记 |
| 2024 | 42-57s | 拉锯白热化——华为加速逼近腾讯，胶着态势 |
| 2025 | 57-69s | 终局——三强趋于均衡，"新一轮 AI 人才争夺"标记 |
| 结尾 | 69-75s | 画面定格，结语"人才是一切竞争力的源头"淡出 |

每章过渡用 0.5s 年份数字放大 + 光晕转场。

## 数据口径

年度招聘总人数（含校招+社招，基于公开财报/新闻报道估算，标注为"估算值"）：

| 年份 | 阿里巴巴 | 腾讯 | 华为 |
|------|---------|------|------|
| 2021 | 88,000 | 78,000 | 45,000 |
| 2022 | 112,000 | 72,000 | 38,000 |
| 2023 | 35,000 | 68,000 | 55,000 |
| 2024 | 42,000 | 52,000 | 85,000 |
| 2025 | 48,000 | 55,000 | 78,000 |

年度数据之间通过 `interpolate` 在季度节点平滑过渡，季度数据为前后两年的加权平均。

## 画面布局

```
┌─────────────────────────────────────────┐
│  年份 / 季度标签       关键事件文字      │ ← 顶部信息栏（高 120px）
│                                          │
│     ┃                                           ┃
│     ┃  [阿 里] ████████████  128,000 人  ┃
│     ┃  [腾 讯] ██████████    89,000 人  ┃     ← 三柱核心区
│     ┃  [华 为] ███████████  102,000 人 ┃
│     ┃                                           ┃
│                                          │
└─────────────────────────────────────────┘
```

- 数字在柱子右侧，等宽数字（Tabular Lining）确保对齐稳定
- 柱子带渐变填充、圆角顶部、内阴影光泽
- 领先柱子发光阴影增强

## 文件结构

```
src/RecruitmentRace/
├── index.tsx              — 主组件，Sequence 编排
├── data.ts                — 招聘数据、关键事件、公司配置
├── types.ts               — TypeScript 类型
├── BarChart.tsx           — 三柱竞速核心组件
├── BarItem.tsx            — 单柱（渐变、发光、数值跳动）
├── Header.tsx             — 顶部年份/事件信息栏
├── GridBackground.tsx     — 背景数据网格线
├── Opening.tsx            — 开场标题 + Logo
├── Ending.tsx             — 结尾定格 + 结语
└── ChapterTransition.tsx  — 年份转场动画
```

## 数据流

```
data.ts (静态数据)
  → index.tsx (useMemo 计算帧→数据点映射)
    → BarChart (接收插值后三家数值)
      → BarItem × 3 (单柱渲染 + 数字动画)
    → Header (年份/事件显示)
    → GridBackground
  ← Sequence 控制章节/转场切换
```

## 动画技术

- 柱子高度：`interpolate` + 弹性 easing (cubic-bezier(0.34, 1.56, 0.64, 1))
- 数字跳动：Odometer 式数字变化
- 发光强度：领先方 boxShadow 随帧 interpolate
- 年份转场：scale 1→1.5→1 + opacity 淡入淡出

## 字体

- 数字：Tabular Lining 等宽数字
- 中文：PingFang SC / Microsoft YaHei，与项目现有字体一致

## 音效（可选）

- 柱子增长：低音脉冲
- 排名交替：轻微"叮"声
- 年份切换：低沉过渡音效

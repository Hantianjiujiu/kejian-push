## 这门课讲什么

AI 时代，技术迭代越来越快，但有一项能力不会被替代——**领导力**。

这是一套面向职场人的系统领导力课程，围绕"从普通员工到一线经理再向上跃升"这条主线，帮你完成职业生涯中回报最高的一步跨越。
